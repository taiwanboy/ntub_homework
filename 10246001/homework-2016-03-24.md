* 請印出在 1 到 2000 中，可以被 4 整除，但不能 100 整除，但又要可以被 400 整除的數字。(潤年計算公式)

```
p (1..2000).to_a.select {
	|number| (number%4 == 0) and !(number%100 == 0 and number%400 != 0) 
}
```
